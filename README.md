## ***Overview***
The Car Dealership Management System simulates real-world dealership operations.
It allows sales associates to:
-Record sales and lease contracts
-Compute monthly payments and total costs
-Manage add-ons like Warranty, Detailing, and Rustproofing
-Append data to contracts_with_headings.csv
-Access Admin tools to view, search, and summarize all contracts

## ***Features***
-Choose Sale or Lease contract type
- Calculates:
- Sale Tax (5%)
- Processing & Recording Fees
- Lease Fee (7%)
- Monthly Payments

  ## ***User Stories***
-As a customer, I want to record a car sale or lease so that I can track all dealership transactions.
-As a sales associate, I want to calculate monthly payments and totals automatically, so I can quote customers quickly and accurately.
-As an administrator, I want to view, search, and filter contracts, so I can manage dealership performance and reports.
-As a buyer, I want to select optional add-ons (Warranty, Detailing, Rustproofing) or the Full Package, so I can customize my purchase.

## ***Future Enhancements***
-Advanced search filters (price, mileage, model)
-Graphical analytics dashboard
